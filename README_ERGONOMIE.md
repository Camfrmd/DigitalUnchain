# Ergonomie

Nous avons le fait le choix d'implémenter des fonctions d'ergonomie dans la partie articles.
- Changer le thème de couleur, pour des utilisateurs necessitant des couleurs différentes (hypersensibilité à la couleur, daltonisme)
- Changer la taille de police, pour des utilisatuers ayant des difficulés de lecture.
- Et enfin la possibilité de changer de police, notamment avec le mode "monospace" plus adapté aux personnes dyslexiques.
